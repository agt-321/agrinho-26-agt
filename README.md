# RaizLab - Agrinho Programação 2026

Projeto autoral da subcategoria 3 do Agrinho Programação 2026. A proposta é transformar o tema biodiversidade do solo em uma central interativa de decisão, com núcleo visual do solo, leitura de talhões, práticas de recuperação e quiz.

## Objetivo

Mostrar como cobertura vegetal, rotação de culturas, planejamento do tráfego de máquinas, sensores, drones, IA e IoT ajudam a proteger a vida do solo contra erosão, compactação, excesso químico e solo descoberto.

## Como usar

1. Abra `index.html` no navegador.
2. Escolha um risco no painel "Risco observado no talhão".
3. Veja o núcleo do solo mudar junto com as leituras de vida, água e poros.
4. Compare práticas de recuperação na seção "Plano de recuperação".
5. Clique nos talhões do mapa e responda ao quiz final.

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 com Grid, Flexbox, animações, transições e Media Queries
- JavaScript puro para manipulação do DOM
- Sem bibliotecas e sem frameworks

## Mídias e autoria

- Imagem em `assets/img/solo-vivo-hero.webp`, gerada com ferramenta de IA da OpenAI/Codex a partir de prompt autoral sobre solo vivo, raízes, sensores e drone, com o PNG original preservado na pasta de imagens.
- Identidade visual, textos, núcleo do solo, mapa de talhões e interações foram criados de forma autoral para o projeto.
